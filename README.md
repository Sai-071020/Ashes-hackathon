## Getting Started

Simply open `index.html` in your web browser. No build process, no dependencies, no installation required!

## Sections

1. **Hero Section** - Eye-catching header with countdown timer and CTA buttons
2. **About Section** - Information about the hackathon
3. **Themes Section** - Four problem statement cards with hover effects
4. **Timeline Section** - Event schedule in timeline format
5. **Prizes Section** - Prize cards with amounts and descriptions
6. **How It Works** - Step-by-step process breakdown
7. **FAQ Section** - Accordion-style frequently asked questions
8. **Footer** - Contact information and social links

## Customization

- **Countdown Date**: Edit the `hackathonStartDate` variable in `script.js`
- **Colors**: Modify CSS variables in `:root` section of `style.css`
- **Content**: Update HTML content in `index.html`

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
